Megaloader168.hex - compact loader for atmega168. Use fuses: Low=DE, High=D7, Extended=FA.
Megaloader328.hex - compact loader for atmega368. Use fuses: Low=DE, High=DC, Extended=FD.
Megaloader site - http://www.microsyl.com/index.php/2010/03/30/megaload/

oerxXXtiny168.hex - programm for original Expert Tiny/2G LRS  reciever module
oerxXXorange328.hex - programm for Open/Orange LRS reciever module 

Recomendations:

You may use Arduini or Megaload bootloader for progects on Atmegs328.
You must use Megaload bootloader for Atmega168, becouse default adduino loader is too large.
